📐 Математика: Composite Final Score (CFS)

1️⃣ Базовая Вирусность (VI)

VI = views + reactions × 5 + replies × 3 // Реакции важнее просмотров в 5× // Комментарии — сильный сигнал вовлечения
viewsreactions ×5replies ×3

2️⃣ Channel EMA Memory (сглаживание)

EMA = α × VI + (1 - α) × prev_EMA α = 0.3 // коэффициент сглаживания // Накопительная история канала // Первый пост: EMA = VI
накопительныйα=0.3JSON хранилище

3️⃣ Relative Virality Index (RVI)

RVI = VI / max(EMA, 1) // Насколько пост аномален для канала // RVI=3 → пост в 3× лучше среднего // Уравнивает большие и малые каналы
аномалиямасштабинвариант

4️⃣ Size Multiplier (нормализация)

sizeM = min(subs / medianSubs, 3.0) medianSubs = median(all channel subs) // Бонус крупным каналам, но не более ×3 // Малый канал с 50K subs → sizeM≈0.5
cap ×3медиана нормировка

5️⃣ Freshness (давность публикации)

age = (now - postDate) / 3600 // часы window = hoursToCheck × 2 freshness = max(0, 1 - age / window) // Старый пост → freshness → 0 // Свежий (0h) → freshness = 1.0
временной штрафлинейный спад

6️⃣ Composite Final Score (CFS)

CFS = RVI × sizeM × (0.7 + 0.3 × freshness) // freshness даёт 70%-100% веса // Пример: RVI=5 × sizeM=2 × 0.85 = 8.5 // Публикуем только CFS > adaptiveThreshold
итоговый ранг70% freshness floor

7️⃣ Adaptive Threshold

mean = avg(all CFS scores) threshold = mean × 1.5 // Топ ~33% публикаций проходит // Автоматически масштабируется // с качеством выборки
динамическийmean ×1.5

8️⃣ Repost Discovery Score

repostCount += 1 // каждый раз когда виден // Каналы с repostCount > 1 // показываются первыми в UI // Сигнал: канал репостят часто // → вероятно качественный источник
накопительныйприоритет в UI

🔍 Анти-Баян: pHash алгоритм

Алгоритм перцептивного хэширования

1. resize(img, 8×8) → grayscale 2. compute DCT (discrete cosine transform) 3. take top-left 8×8 of DCT 4. mean = avg(64 values) 5. hash = bits where pixel > mean 6. distance = hamming(hash1, hash2) 7. isDupe = distance < threshold(10)

Хранилище хэшей

phash_db.json: [ { id, channel, hash, date }, ... ] // JSON вместо SQLite // совместим с любой платформой // hamming distance < 10 → баян
